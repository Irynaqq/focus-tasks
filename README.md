# Focus Tasks — React App with Specialized Libraries

## Опис проєкту
**Focus Tasks** — це навчальний React-додаток для керування списком завдань
із режимом фокусу. Проєкт демонструє інтеграцію спеціалізованих бібліотек,
які покращують користувацький досвід та розширюють можливості React-додатку.

Додаток дозволяє:
- додавати та видаляти завдання;
- позначати завдання як виконані;
- очищати виконані завдання;
- вмикати режим фокусу;
- отримувати сповіщення про дії користувача та бездіяльність.

---

## Використані технології та бібліотеки
- **React** + **Vite**
- **React Icons** — для візуальних іконок інтерфейсу
- **React Toastify** — для відображення сповіщень (успіх, помилка, попередження)
- **React Idle Timer** — для відстеження бездіяльності користувача
- **CSS (без UI-фреймворків)** — для стилізації інтерфейсу

---

## Функціональність бібліотек

### React Icons
Використовується для іконок кнопок (додавання, видалення, очищення, режим фокусу),
що робить інтерфейс більш зрозумілим та зручним.

### React Toastify
Застосовується для показу повідомлень:
- успішне додавання завдання;
- видалення завдання;
- попередження про помилки;
- сповіщення про зміну режиму фокусу;
- повідомлення при бездіяльності користувача.

### React Idle Timer
Використовується для визначення періодів бездіяльності користувача.
При бездіяльності відображається відповідне сповіщення,
а при поверненні активності — повідомлення про відновлення роботи.
